# Brand Assets

Place the following files here for HACS:

- `icon.png` — 256×256px integration icon (required for HACS default store)
- `logo.png` — wider logo variant (optional)

For a custom repository (not HACS default store), these are optional.
A suitable EcoFlow icon can be found at:
https://brands.home-assistant.io/ (if/when submitted to HA brands)

For now, HACS will show a generic icon without these files.
